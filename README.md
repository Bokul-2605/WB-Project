📌 Overview

West Bengal Explorer is a frontend web project that showcases the culture, heritage, and beauty of the Indian state West Bengal.
The project is designed to provide users with a visually engaging and informative experience using modern web technologies.

✨ Features
🎨 Clean and responsive UI design
🏞️ Highlights of West Bengal’s culture and landmarks
📱 Mobile-friendly layout
⚡ Fast and lightweight frontend implementation
🌐 Deployed using GitHub Pages
🛠️ Tech Stack

CSS3 – Styling & layout
JavaScript – Interactivity

🍛 Unique cuisine
